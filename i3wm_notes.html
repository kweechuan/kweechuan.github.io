My I3WM notes

As a long time command line user, decided to give I3WM a go.

What I have:

  1. A used SSD 256GB for my old (but still very functional) laptop 2570p.
  2. A laptop caddy for the SSD.

Ubuntu server 22.04 installation (without network)

No surprises

Install I3WM

      apt-get install i3

Desktop Manager with lightdm

      sudo apt-get install lightdm

Wifi

Disable netplan, use NetworkManager

       cd /etc/NetworkManager/conf.d/
       vim manage-all.conf
       sudo systemctl disable --now systemd-networkd.service systemd-networkd.socket networkd-dispatcher.service && sudo systemctl restart NetworkManager
       systemctl enable NetworkManager
       reboot

Audio

      systemctl --user enable pulseaudio
      systemctl --user start pulseaudio

Timezone

      sudo timedatectl set-timezone Asia/Singapore

Trackpad Natural Scrolling

     
       cd /usr/share/X11/xorg.conf.d/
       vim 40-libinput.conf
